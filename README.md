# Alimentador-
Este trabalho apresenta o desenvolvimento de um sis tema digital para alimentação automática de cães, projetado em nível  RTL e implementado em VHDL. O processo abrange a definição da  arquitetura RTL, a elaboração da HLSM, a obtenção do datapath e  sua implementação em VHDL. O sistema tem como objetivo liberar  ração nos horários programados.
